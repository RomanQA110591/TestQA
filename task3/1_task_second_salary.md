# Задача 1: Вторая зарплата

## Условие
Дана таблица `Employee`:
- `id` INT PRIMARY KEY
- `salary` INT

Необходимо написать SQL-запрос, который возвращает **вторую по величине уникальную** зарплату.  
Если такой зарплаты нет (например, все зарплаты одинаковы или записей меньше двух), результат должен быть `NULL`.

## Решение (PostgreSQL)

-- Используем DISTINCT для уникальных значений, сортируем по убыванию,
-- берём первую строку со смещением 1 (вторая по счёту).
-- Если строк нет, подзапрос возвращает NULL автоматически.

SELECT (
    SELECT DISTINCT salary
    FROM Employee
    ORDER BY salary DESC
    LIMIT 1 OFFSET 1
) AS SecondHighestSalary;

*Пояснение*

DISTINCT - гарантирует уникальность зарплат.

ORDER BY salary DESC – сортировка от большей к меньшей.

LIMIT 1 OFFSET 1 – пропускаем первую (максимальную) и берём следующую.

Внешний SELECT оборачивает подзапрос, чтобы при отсутствии строк результатом было NULL (вместо пустого набора).
